VSC settings

1- create a dir named ".vscode"
2- create a file named "settings.json"
3- set the body to :
{ "editor.defaultFormatter": "esbenp.prettier-vscode", "editor.bracketPairColorization.enabled": true, "editor.formatOnSave": true, "editor.formatOnPaste": true, "editor.wordWrap": "on", }

prettier configs

1- create a file named ".prettierrc"
2- set the body to :
{ "singleQuote": true, "printWidth": 200, "proseWrap": "always", "tabWidth": 4, "useTabs": false, "trailingComma": "none", "bracketSpacing": true, "jsxSingleQuote": false, "semi": true }
